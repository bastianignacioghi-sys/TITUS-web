## 10. ANIMACIONES GLOBALES — Lo que separa un sitio amateur de uno profesional

PROBLEMA: No hay animaciones de scroll o son bruscas.
CORRECCIÓN — implementa con IntersectionObserver:

```javascript
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
    }
  });
}, { threshold: 0.15 });

document.querySelectorAll('.animate-on-scroll').forEach(el => observer.observe(el));
```

CSS base:
```css
.animate-on-scroll {
  opacity: 0;
  transform: translateY(40px);
  transition: opacity 0.7s ease, transform 0.7s ease;
}
.animate-on-scroll.visible {
  opacity: 1;
  transform: translateY(0);
}
```

- Agrega la clase .animate-on-scroll a: títulos de sección, cards de portafolio, filas de servicios, contadores
- Los elementos hijos dentro de una sección deben tener delays escalonados:
  .animate-on-scroll:nth-child(1) { transition-delay: 0s; }
  .animate-on-scroll:nth-child(2) { transition-delay: 0.1s; }
  .animate-on-scroll:nth-child(3) { transition-delay: 0.2s; }
